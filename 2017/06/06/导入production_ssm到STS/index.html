I added src / main / resources folder to classpath to solve the error.

Select the project and right click > Build Path - Configure Build Path Click> Java Build Path - Add Folder Click> Source Folder Selection - resources folder Check - OK Click

– Must be linked with DB to match src/main/resources/jdbc.properties file
jdbc.driverClassName=com.mysql.jdbc.Driver
jdbc.url=jdbc\:mysql\://localhost\:3306/production_ssm
jdbc.username=root
jdbc.password=1234

– Run a successful web server
Finally, the web server normally loads and runs the production_ssm project.

– PreCautions:
I initially thought the web server would work well
when the Logger record was shown in the console window.
If the process of linking DB is not done, the web server does not operate normally.
